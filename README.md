# .NET WPF CRUD Application

A desktop CRUD application built with C# and WPF demonstrating user management functionality including registration, password recovery and basic data handling. The project illustrates fundamental desktop application architecture and UI development in the .NET ecosystem.

## Features

- Create, Read, Update and Delete (CRUD) operations  
- User registration  
- Password recovery ("Forgot password")  
- Desktop user interface built with WPF  
- Basic data management and validation  

## Technologies

- C#
- .NET
- WPF (Windows Presentation Foundation)
- XAML

## Purpose

This project demonstrates how a desktop CRUD application can be developed using WPF and C#. It highlights basic application structure, user interface design, and data handling in a .NET desktop environment.
